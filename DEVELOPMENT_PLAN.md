# ACF管控系统 - 开发计划

## 项目信息
- **项目名称**：ACF物料全生命周期管控系统
- **开发周期**：8周（2个月）
- **技术栈**：Vue 3 + Spring Boot + MySQL

### Phase 2: 基础数据管理（Week 2）
**目标**：实现料号、标签模板、预警规则等基础配置功能

   - 模板CRUD
   - 字段配置（LOT、型号、有效期等）

3. **LOT号规则管理**
   - 生成规则配置

4. **预警规则管理**

## 技术架构

### 后端技术栈
- **框架**：Spring Boot 3.2
- **ORM**：MyBatis-Plus 3.5
- **数据库**：MySQL 8.0
- **缓存**：Redis 7.0
- **定时任务**：Quartz 2.3
- **文档**：SpringDoc OpenAPI 3

### 前端技术栈
- **框架**：Vue 3.4 + TypeScript
- **UI库**：Element Plus 2.5
- **状态管理**：Pinia 2.1
- **路由**：Vue Router 4.2
- **图表**：ECharts 5.4
- **HTTP**：Axios 1.6
- **构建工具**：Vite 5.0

### 开发工具
- **IDE**：IntelliJ IDEA + VS Code
- **版本控制**：Git
- **接口测试**：Postman/Apifox
- **数据库工具**：Navicat/DataGrip

## 数据库表结构

### 1. 基础数据表
```
- acf_material (料号表)
- acf_label_template (标签模板表)
- acf_lot_rule (LOT号规则表)
- acf_alert_rule (预警规则表)
```

### 2. 业务表
```
- acf_lot (LOT号主表)
- acf_inventory (库存表)
- acf_transaction (出入库记录表)
- acf_operation_log (操作日志表)
```

### 3. 系统表
```
- sys_user (用户表)
- sys_role (角色表)
- sys_menu (菜单表)
- sys_log (系统日志表)
```

## 开发规范

### 代码规范
- 后端遵循阿里巴巴Java开发规范
- 前端遵循Vue官方风格指南
- 统一使用UTF-8编码
- Git提交信息规范

### 接口规范
- RESTful API设计
- 统一响应格式
- 统一异常处理
- 接口版本控制

### 数据库规范
- 表名小写下划线分隔
- 字段名小写下划线分隔
- 必须有主键、创建时间、更新时间
- 软删除字段deleted
